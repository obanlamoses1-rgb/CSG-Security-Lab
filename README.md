# CSG Security Lab 🛡️

Welcome to my cybersecurity home lab.

I created this lab to build practical skills in cybersecurity, networking, and system administration through hands-on practice.

This lab is built using VMware and includes Windows and Linux environments to understand how systems communicate, how they are configured, and how security concepts are applied.

## Lab Environment

### Virtual Machines

- Windows 10
  - Used as an endpoint machine for Windows administration and security practice.

- Windows Server
  - Used for server management and future Active Directory practice.

- Kali Linux
  - Used for cybersecurity tools, testing, and security learning.

## Skills Practiced

- VMware virtual machine setup
- NAT and Host-only networking
- Windows installation and configuration
- Linux environment setup
- Network troubleshooting
- Cybersecurity lab documentation
